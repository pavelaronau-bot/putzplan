# ADR-0005. Node.js 20 и argon2id для первого среза

**Дата:** 20.07.2026 · **Статус:** замещено ADR-0007 (Sprint 1)

## Контекст

> Решение относилось к каркасу Gate 1. В Sprint 1 утверждён стек
> FastAPI + Python 3.12 — см. ADR-0007.

Нужен рабочий каркас бэкенда, который можно запустить и проверить тестами
уже сейчас, не привязываясь к тяжёлому фреймворку на этапе Gate 1.

## Решение
Node.js 20, минимальный HTTP-слой на `node:http`, `pg` для базы,
`@node-rs/argon2` для паролей (argon2id, 19 МиБ, 2 итерации, параллелизм 1 —
рекомендация OWASP). Структурные логи в JSON с `request_id`.

## Последствия
- Каркас запускается и тестируется без Docker и внешних сервисов.
- При росте функциональности слой маршрутизации заменяется на NestJS или
  Fastify без изменения слоёв `db`, `rbac`, `audit`.
- `@node-rs/argon2` поставляется с предсобранными бинарниками — сборка
  в CI не требует компилятора.
- Подпись access-токена сейчас HMAC с секретом из переменной окружения;
  в production ключ переносится в KMS с ротацией раз в квартал.
